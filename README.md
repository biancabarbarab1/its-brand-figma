# its-brand-figma
O Projeto foi desenvolvido para a criação de uma loja de moda streetwear e oversized, pensado para um cliente real que deseja lançar seu e-commerce online.

O objetivo do projeto é criar uma experiência moderna, intuitiva e visualmente marcante, focada no público jovem e conectado ao lifestyle urbano.

Sobre o projeto

A IT’S BRAND é uma loja digital com foco em três principais estilos:

Oversized

Streetwear

Lifestyle de verão (coleção praia exclusiva)

O design foi pensado para transmitir estilo, exclusividade e identidade, utilizando uma paleta sofisticada com tons de vinho escuro, preto e branco.


Experiência do usuário

O aplicativo foi estruturado para oferecer uma navegação simples e fluida, permitindo que o usuário:

Acesse rapidamente sua conta (login e cadastro)

Explore coleções através da tela inicial

Navegue por categorias específicas

Visualize produtos em destaque através de carrossel de lançamentos


Funcionalidades do e-commerce

O projeto contempla as principais funcionalidades de uma loja online completa:

Visualização detalhada dos produtos
(imagem, descrição, preço, tamanhos disponíveis)

Sistema de carrinho de compras

Aplicação de cupons de desconto

Destaques promocionais (ex: cashback e ofertas)

Processo de checkout simplificado

Rastreamento de pedidos

Tela de histórico de compras

Navegação por categorias

Sistema de favoritos

Acesse páginas individuais de produtos


Objetivo do design

Criar uma interface que una:

Estética moderna e urbana

Facilidade de uso

Organização de informações

Experiência agradável durante toda a jornada de compra


Este projeto representa o desenvolvimento de um e-commerce completo, desde a identidade visual até a estrutura da experiência do usuário, com foco em criar uma marca forte e uma navegação intuitiva.
